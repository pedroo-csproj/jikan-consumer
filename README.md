# Jikan Consumer
> Project to consume the Jikan API an unofficial MyAnimeList API.

## Technologies Used
- [Go](https://go.dev/);
- [Mux](https://github.com/gorilla/mux/);
- [Jikan](https://jikan.moe//);

## Features
List the ready features here:
- Retrieve Anime;
- Retrieve Anime Statistics;
- Retrieve a Club;
